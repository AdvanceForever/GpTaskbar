# GpTaskbar
中国股票悬浮窗及时查看股票状态。调用新浪接口查询。每5秒刷新一下数据。可悬浮在任务栏上，自动检测悬浮状态，被覆盖后自动跳到最前。

## 技术栈
开发语言C#，.Net10.0.3 运行环境。下载体积大于100MB表示包含运行环境，无需单独安装。

## 股票代号填写
填写股票代号示例: sh000001 (上证指数), sz399001 (深证成指), sh600036 (招商银行)，自动即时获取交易状态。
前缀说明：sh（上证），sz（深圳）。

## 更新间隔
默认5秒更新一次数据，新浪接口最小5秒。

## 浮窗显示
在配置选项中可自定义显示字段，股票代码，股票名称，当前价格，涨跌差价，涨跌幅度。
